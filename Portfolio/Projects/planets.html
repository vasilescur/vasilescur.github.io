GRAVITATIONAL PLANETS

An N-body simulation written in Java with Processing

Gravitational Planets is a simulation of an "N" number of bodies in a two dimensional space, rotating and moving around according to the Newtonian rules of gravity and physics.

Although there are no collision mechanics and there is no current way to interact with the simulation, these are both planned features on which I will continue to work in the future. Check back soon!
