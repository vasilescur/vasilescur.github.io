BrainPuddle
Programming, made way too complicated.

What is BrainPuddle?

BrainPuddle is a small programming language similar to assembler. It includes an easily expandable array of features, and this makes it quite powerful despite its obscurity, inclarity, arcane aspect, and the fact that it's completely pointless.

The Story Behind BrainPuddle

A while ago, I went to a wonderful little event called a Hackathon. This consisted of a 24-hour time slot in which small groups could work continuously to create a project of some sort, using programming and computers.

My group consisted of myself and a friend of mine, Emily Kiehl. Together, we got really bored, so I decided to write a programming language from scratch, in C++. Several RedBulls, tons of frustrating bugs, and 20 hours later, BrainPuddle was brought into the world.

BrainPuddle: The Language

The syntax of BrainPuddle itself is quite simple. In essence, it is a series of primitive operations performed upon a "selected" value. To perform an addition, for example, the code would be as follows:


DIM A     // Declare a variable called A
SLC A     // Set the selected variable to A
GET       // Read a value from the keyboard

DIM B     // Same as before, but using B
SLC B
GET

SLC A
ADD B     // Add the value of the selected var (A) into B
SLC B

OUT      // Output the value of the selected var (B)


BrainPuddle also supports looping and conditionals. However, a loop can only be realized using a Label/Goto structure, like this:


LBL A
SAY Hello, World!  // Outputs "Hello, World!"
JMP A              // "Jump" to A

Finally, BrainPuddle also includes conditionals, with a basic structure of _IF [condition] [target], in which it compares the selected variable to the target using the condition (GRE = greater than, LSS = less than, EQL = equal to, etc.). All conditionals must end with END. Conditionals can be nested up to 10 levels, after which there is a significant performance drop.

Getting Started with BrainPuddle

Getting started is easy. Just click the button below to be taken to the BrainPuddle Github page. There, you will find all of the source files used in the project. Simply download them and compile. To write a BrainPuddle program, just make a text file with a .pdl extension and write your code inside. Then, put the compiled BrainPuddle.exe in the same folder as your program and run it. Type the name of your program (don't forget the file extension), and you're good to go.
